# display 常用值

| 属性值       | 作用                                                         |
| :----------- | :----------------------------------------------------------- |
| none         | 元素不显示，并且会从文档流中移除                             |
| block        | 块元素类型。默认宽度为父元素宽度，可设置宽高，换行显示       |
| inline       | 行内元素类型。默认宽度为内容宽度，不可设置宽高，不能设置垂直方向的padding和margin，同行显示 |
| inline-block | 行内块元素类型。默认宽度为内容宽度，可以设置宽高，同行显示   |
